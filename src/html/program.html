Во время игры вас ждёт:

  • ДЕНЬ 1.

    • - Знакомство с синтаксисом языка
    • - Создаем свой первый класс
    • - Научимся различать функции и методы
    • - Установим программу редактора кода и запустим там свой код

  • ДЕНЬ 2.

    • - Поработаем с массивами
    • - Вычислим медианные значения
    • - Научимся определять минимальные и максимальные значения в массивах данных

  • ДЕНЬ 3.

    • - Запустим свой первый сервер
    • - Научимся работать с фреймворком Flask
    • - Заканчиваем первый личный проект на Python

  • БОНУСЫ:

    • - Обучающие видео "Python-интерпретатор" и "Редактор кода VSC"
    • - А также видеоинструкция о постановке целей для себя и своего стартапа - методология, которую используют Google, Linkedin, Oracle и многие другие.

Для игры мы разработали специальный тренажер на Python, в котором вы изучите теорию, получите практические задания и сможете автоматизированно проверить свой код.

Выполнив все задания тренажера, вы создадите свой первый IT-проект — небольшую CRM-систему.
